# rekar
## 1. clone project
```
git clone https://github.com/kak-smko/rekar
```

## 2. Modify config directory files 

## 3. install backend
```
pip install -r requirements.txt
python command.py extension
``` 
## 4. run backend
```
python index.py
```

## 5. install frontend
```
cd frontend
npm i
```

## 6. run Frontend
```
cd frontend
npm run serv
``` 

open browser and go to localhost:8080
